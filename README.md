# Bytecoin 2.0

Bytecoin 2.0 is a groundbreaking cryptocurrency project that employs state-of-the-art cryptographic techniques and consensus algorithms to ensure secure and efficient transactions.

## Ed25519 Cryptography
Ed25519 is a public-key signature system with several attractive features:
- High performance: It is optimized for speed and efficiency.
- Strong security: Provides protections against various cryptographic attacks.
- Simplicity: Easy to implement correctly, reducing the chances of vulnerabilities.

## CryptoNight Consensus Algorithm
Our project utilizes the CryptoNight consensus algorithm, which provides:
- **ASIC resistance**: Allows for mining on consumer hardware, promoting decentralization.
- **Privacy**: Ensures that transaction details remain confidential.

## Project Structure
- **/src**: Contains the core code for the cryptocurrency.
- **/tests**: Comprehensive unit tests to validate functionality.
- **/docs**: Documentation related to the project.

## Building Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/duicoin14/duicoin-core.git
   cd duicoin-core
   ```
2. Install the dependencies:
   ```bash
   sudo apt-get install build-essential libtool autotools-dev automake pkg-config
   ```
3. Build the project:
   ```bash
   make
   ```
